# 1. ChessMaster ♟️ - Premium Full-Stack Chess Platform

## 📌 Project Overview
ChessMaster is a high-performance, feature-rich chess application designed for both casual play and deep competitive analysis. It offers a seamless cross-platform experience (Web, iOS, Android) with premium aesthetics and real-time interaction.

## 🌟 Key Features
- **Real-time Multiplayer**: Global matchmaking and private room challenges powered by Socket.io.
- **AI Opponent**: Multiple difficulty levels with customizable board settings, integrated directly in the browser.
- **Local Play**: Pass-and-play mode for over-the-board games with friends.
- **Chess960 (Fischer Random)**: Full support for Chess960, including specialized castling logic and position generation.
- **Game Review Engine**: A dedicated analysis suite to identify "Blunders," "Mistakes," and "Brilliant" moves using an integrated chess engine.

## 🛠️ Technical Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS (v4), Zustand, Three.js, Framer Motion.
- **Backend**: Node.js, Express (v5), Socket.io, Prisma, PostgreSQL, JWT.
- **Infrastructure**: Vercel (Web), Capacitor (Mobile).

# 2. CyberShell Hybrid 🚀 - Advanced C-Shell with AI & Cyberpunk GUI

## 📌 Project Overview
CyberShell Hybrid is a two-tier shell architecture that bridges the gap between low-level system operations and modern AI-enhanced user experiences. It features a high-performance Linux-based C engine wrapped in a stunning, immersive PyQt6 interface with integrated AI assistance.

## 🌟 Key Features
- **Linux C Engine**: Handles process forking, memory management, and kernel interaction via standard POSIX syscalls.
- **Pipeline & Redirection**: Full support for command chaining (`|`) and I/O redirection (`<`, `>`, `>>`).
- **Integrated AI Side-panel**: Live chat powered by Google Gemini for real-time command assistance and general queries.
- **Cyberpunk Aesthetic**: Translucent, frameless windows with neon glow effects, custom backgrounds, and themeable audio.
- **System Dashboard**: Live monitoring for CPU, Memory usage, and system Uptime.

## 🛠️ Technical Stack
- **Backend Logic**: C (GCC Compiler), POSIX Syscalls.
- **Frontend Layer**: Python 3, PyQt6, Markdown.
- **AI Integration**: Google Gemini API.
- **Monitoring**: Psutil, Winsound.

# 3. NuGames 🕹️ - Java-Based 2D Game Suite

## 📌 Project Overview
NuGames is a collection of classic and original 2D arcade games developed entirely in Java. The project showcases core game development principles, including custom rendering engines, collision logic, and real-time multiplayer mechanics, all built without external game frameworks.

## 🌟 Key Features
- **Cross to Kacha**: A fast-paced "Frogger-style" competitive game where players cross a road to collect samosas and return to safety.
- **Classic Remakes**: Includes Flappy Bird, Ludo Star, Ping Pong, and Star Ship.
- **Custom Game Loops**: Optimized loops for smooth 60 FPS rendering.
- **Collision System**: Precise AABB collision detection built from scratch.
- **Multiplayer Support**: Local 1v1 competitive mode with concurrent input handling.

## 🛠️ Technical Stack
- **Language**: Java SE.
- **Graphics & UI**: Java AWT and Swing for canvas-based rendering.
- **Paradigm**: Object-Oriented Programming (OOP).

# 4. NuQuizzer ❓ - Python-Powered Competitive Quiz Platform

## 📌 Project Overview
NuQuizzer is an interactive, multi-mode quiz application designed for high-speed knowledge testing. Developed in "pure Python" with a focus on cross-platform usability, it offers both a focused single-player experience and a high-stakes local multiplayer mode.

## 🌟 Key Features
- **Competitive Quiz Modes**: Features single-player mouse navigation and a unique "split-keyboard" two-player mode.
- **Gamified Scoring**: Advanced scoring logic with rewards and heavy penalties to encourage strategic play.
- **Threaded Timers**: Multi-threaded logic for global clocks and progress bars to ensure smooth UI performance.
- **Session Persistence**: Automated saving of player statistics and "Recent Wins" using CSV storage.

## 🛠️ Technical Stack
- **Language**: Python 3.
- **GUI Framework**: Tkinter + ttk (Themed Tkinter).
- **Audio/VFX**: PIL/Pillow for visuals, custom multi-threading for responsiveness.
- **Data**: CSV-based leaderboard persistence.

# 5. NUsphere 🌐 - Full-Stack Academic Management Dashboard

## 📌 Project Overview
NUsphere is a comprehensive academic productivity platform designed to help students manage their university lifecycle. It integrates course management, attendance tracking, GPA calculation, and grade forecasting into a single, unified dashboard.

## 🌟 Key Features
- **Academic Semester Tracker**: Live progress monitoring with week-by-week status labels and milestone countdowns.
- **Intelligent Attendance Management**: Automated reporting with logic-based zones (Safe/Warning/Danger) and "shortage prediction" to help students maintain high attendance.
- **Grade Prediction Engine**: A weighted forecasting model that estimates final course grades based on mid-semester assessment data (Quizzes, Assignments, PBLs).
- **GPA/CGPA Tools**: Real-time calculators and target estimators allowing students to plan required future performance to reach specific CGPA goals.
- **Role-Based Governance**: Specialized administrative controls for semester initialization and academic window configuration.

## 🛠️ Technical Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Lucide React, React Router.
- **Backend**: Node.js, Express, MongoDB with Mongoose ORM.
- **Authentication**: JWT-based session management and bcrypt security.

## 🚀 Technical Highlights & Challenges Solved
- **Complex Academic Logic**: Developed robust mathematical models for weighted grade prediction and attendance percentage tracking.
- **Integrated Full-Stack Architecture**: Seemlessly connected a modern React SPA with a MongoDB-backed REST API for real-time data persistence.
- **User-Centric Design**: Focused on visual progress indicators and intuitive dashboards to simplify complex academic planning.
